Implicit Binding

Explicit Binding

如果不想在 object 创建函数，我们可以手动把函数的this绑定到对象上。

可以使用bind、call或apply。

call 和 apply 可以直接以 object 作为 argument。
bind 需要指向一个新的变量。

new Binding

创建 class 的 instance 时，this 会指向这个 instance

每个 instance 都会有自己的 context。
